# book2skill 产品变现规划

> 版本：v1.0 · 2026-07-08
> 仓库：[book2skill](https://github.com/lgjjennie-ship-it/book-skill)

---

## 一、产品定位

**一句话：把任意书籍变成可对话的 AI 技能文件。**

用户在 OpenCode 里提到书名，AI 就像读过这本书一样回答。解决的核心痛点是——"我有书，但 AI 训练数据里没有，或者记错了"。

**与竞品的差异：**

| | 直接把 PDF 贴给 ChatGPT | book2skill |
|---|---|---|
| 单次可用性 | 一次对话就丢上下文 | 永久技能，随时调用 |
| 章节精度 | 全文一股脑喂，定位模糊 | 逐章拆解，精确到节 |
| 复用性 | 每次重新上传 | 生成一次，团队共用 |
| 中文书适配 | 目录识别依赖模型能力 | 原生中文章节+子节模式匹配 |

**护城河：** OpenCode 生态技能商店的先发位置。当未来技能商店成型时，book2skill 是"制作技能"这个品类的默认工具。

## 二、目标用户

| 用户画像 | 场景 | 付费意愿 |
|----------|------|:--:|
| **技术个人用户** | 把自己收藏的技术书转成 AI 技能辅助学习/工作 | 低 |
| **知识创作者** | 把自己的专业知识输出成可对话的数字资产出售 | 高 |

| **出版社** | 新书推广——先对话再买书，增加转化 | 高 |

## 三、变现路径

### 第一阶段：流量积累（0-3 个月）

**目标：** 让 OpenCode 用户知道 book2skill 的存在。

**动作清单：**

- [x] GitHub 开源，README 中英文双版
- [x] README 挂 `#opencode` 标签
- [ ] 用 book2skill 生成 5 本公版经典书 skill，开源到独立 repo，README 注明"由 book2skill 生成"
- [ ] 在 OpenCode 社区（Discord / Reddit / GitHub Discussions）发帖：
  - "我把《孙子兵法》变成了可对话的 AI 技能"
  - "5 秒把一本书转成 OpenCode 技能——book2skill 使用体验"
- [ ] 即刻 / 小红书发中文教程帖："手把手把你的电子书变成 AI 知识库"
- [ ] GitHub 仓库设置 SEO topic：`opencode`、`ai-skills`、`book-to-skill`、`python`、`ebook`

**关键指标：** GitHub stars 破 100，社区帖子总互动破 1000。

---

### 第二阶段：个人变现（3-6 个月）

**目标：** 从工具作者变成技能资产卖家。

**路径 A：知识资产化**

1. 选定一个你擅长的细分领域（候选：AI 提示词工程、独立开发者出海、跨境电商选品）
2. 写一本 2-4 万字的实战电子书（不要追求厚度，追求密度）
3. 用 book2skill 把书转成 skill
4. 打包出售："电子书 + AI 对话技能"套装，定价 $49-$99
5. 渠道：Gumroad / Lemonsqueezy / 即刻商店

**为什么比单纯卖电子书贵？** 用户买的不只是文字，是"能和你的知识对话"的体验。

**路径 B：技能定制服务**

1. 在 GitHub README 和社区帖子中标注"接定制需求"
2. 定价：按字数，¥500-5000/本
3. 交付：24 小时内交付完整技能目录 + 部署指导
4. 目标客户：有内部文档但无人维护的中小企业

**路径 C：Affiliate 管道**

在 README 和相关帖子中挂 OpenCode referral（如果 OpenCode 有推荐计划），以及 Calibre 等上下游工具 affiliate。

**关键指标：** 第一笔收入入账（无论金额）。

---

### 第三阶段：规模化（6-12 个月）

**目标：** 从一人作坊变成可复制模式。

**方向 A：技能商店**

搭建 book2skill.io 网站：
- 用户上传 EPUB → 云端转换 → 预览 → 下载 / 安装
- 免费层：每月 3 本
- Pro 层：$9/月，无限转换 + 批量 + 优先解析精度
- 增值服务：人工校对章节结构（¥100/本）

**方向 B：企业服务**

- 打包 book2skill + 部署 + 培训 = ¥10,000-50,000/单
- 使用场景：内部 SOP 文档 → 新员工入职 AI 助手
- 目标：律所、会计所、连锁零售培训部门

**方向 C：出版社合作**

- 新书上市 = 同步上线"AI 对话版"试读章
- 读者和 AI 聊完前两章 → 想继续读 → 导流购买完整版
- 出版社按转化付费，按引流付费，或按 skill 下载量付费

**关键指标：** MRR > ¥10,000。

## 六、风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 版权侵权 | 法律风险，平台封号 | 只做自有内容 + 公版书 + 客户授权。README 显眼处标注版权声明。 |
| OpenCode 技能加载机制变化 | 产品失效 | 工具本身是格式转换器，可适配其他 AI 工具（Cursor/Copilot/Cline） |
| 竞品出现 | 价格战 | 先发优势 + 中文书目录识别壁垒 + 社区生态绑定 |
| 用户不会用命令行 | 转化率低 | v0.2 上拖拽界面 / Electron 打包好的 GUI app |
